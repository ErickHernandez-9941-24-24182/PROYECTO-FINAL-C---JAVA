## Descripción

Este proyecto consiste en una red social académica universitaria desarrollada en **C++**, donde se administran estudiantes, conexiones entre usuarios, proyectos académicos y un ranking académico.  
También se incluye un módulo separado en **Java** para realizar un benchmark comparativo usando estructuras nativas del lenguaje.

El objetivo principal es aplicar estructuras de datos como Hash, AVL, Grafo, Lista Enlazada, Cola y Pila, además de comparar el rendimiento entre estructuras manuales en C++ y estructuras nativas en Java.
